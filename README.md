# Smart Shopping List

Мобильное приложение для ведения списка покупок: добавление позиций, отметка купленного, удаление по одной и массовая очистка.

---

## Описание

| Возможность | Описание |
|-------------|----------|
| Список покупок | Добавление и хранение товаров |
| Статус | Отметка купленных позиций |
| Управление | Удаление одного элемента или очистка списка |

## Установка

### Вариант A — готовый APK

1. Откройте [**Releases / Actions**](https://github.com/fpmi-pmvs2026/pmvs10b-project-oranges/actions) репозитория.
2. Скачайте APK после успешной сборки CI.

### Вариант B — сборка из исходников

1. Клонируйте репозиторий:

```bash
git clone https://github.com/fpmi-pmvs2026/pmvs10b-project-oranges.git
cd pmvs10b-project-oranges
```

2. Откройте проект в **Android Studio**.
3. Соберите и установите приложение на устройство или эмулятор (**Run**).

---

## Использование

| Действие | Результат |
|----------|-----------|
| Нажать **+** | Добавить товар в список |
| Нажать чекбокс у товара | Отметить как купленный / снять отметку |
| Долгое нажатие на элемент | Удалить один товар |
| Кнопка **«Очистить купленные»** | Удалить все отмеченные купленными |
| Кнопка **«Очистить всё»** | Полностью очистить список |
